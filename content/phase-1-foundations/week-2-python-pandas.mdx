## 🗺️ Overview
Deep dive into data manipulation, cleaning, and exploratory data analysis using Pandas to prepare analytical datasets. 

## 🧠 Key Concepts
- **DataFrame** — 2-dimensional labeled data structure.
- **Series** — 1-dimensional labeled array.
- **Filtering & Selection** — Using .loc[] and .iloc[].
- **Aggregation** — Grouping data using .groupby().
- **Handling Missing Data** — Strategies for dealing with NaN values (dropna, fillna).

## 💻 Code Examples
```python
import pandas as pd

# Load and clean data
df = pd.read_csv('orders.csv')
df['shipping_cost'] = df['shipping_cost'].fillna(df['shipping_cost'].mean())

# Aggregation operations
summary = df.groupby('warehouse_id').agg({
    'order_id': 'count',
    'total_amount': 'sum'
}).rename(columns={'order_id': 'total_orders'})

print(summary)
```

```python
# Merging Datasets
customers = pd.DataFrame({'customer_id': [1, 2], 'name': ['Alice', 'Bob']})
merged = pd.merge(df, customers, on='customer_id', how='left')
```
