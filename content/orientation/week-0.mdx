## Welcome

You're about to start a 16-week journey into production-grade Data Engineering. This orientation gets your environment ready and introduces the core landscape.

> Before we write a single pipeline, let's make sure your machine can talk to Python.

## What is Data Engineering?

Data Engineering is the discipline of building the **infrastructure** that lets data flow from raw sources into actionable insights.

| Role | Focus | Core Tools |
|------|-------|------------|
| **Data Engineer** | Pipelines, ETL, infrastructure | Python, SQL, Spark |
| **Data Analyst** | Reports, dashboards | Excel, Power BI, SQL |
| **Data Scientist** | Models, ML | Python, R, TensorFlow |

## Key Concepts

- **ETL/ELT** — Extract, Transform, Load paradigms. The backbone of every pipeline.
- **Data Lake** — Raw, unstructured storage at scale.
- **Data Warehouse** — Structured, optimized storage for analytical queries.
- **Compute vs Storage** — Modern cloud separation enabling independent scaling.
- **Databricks** — Unified analytics platform built on Apache Spark.
