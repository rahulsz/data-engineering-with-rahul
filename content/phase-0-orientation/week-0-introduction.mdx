## 🗺️ Overview
An introduction to the data engineering landscape, the role of a data engineer vs analyst, and setting up the core environment. Understanding these concepts is crucial for building robust data architectures.

## 🧠 Key Concepts
- **ETL/ELT** — Extract, Transform, Load (or Extract, Load, Transform) paradigms.
- **Data Lake** — Storage repository that holds a vast amount of raw data in its native format.
- **Data Warehouse** — Central repository of integrated data from one or more disparate sources.
- **Compute vs Storage** — Modern cloud separation allowing scalable processing independently from data holding.
- **Databricks** — Unified analytics platform built around Apache Spark.
