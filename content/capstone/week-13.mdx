## Overview
Leverage GenAI tools for SQL generation, code scaffolding, and documentation while finalizing your career portfolio. AI acts as your powerful co-pilot, driving productivity and unlocking scale in modern data teams.

## Key Concepts
- **LLMs for Code Generation** — Using ChatGPT or GitHub Copilot to scaffold PySpark or SQL code.
- **Prompt Engineering for Data** — Being specific about schema and dialect when asking for queries.
- **Automated Documentation** — Feeding scripts into AI to generate robust docstrings and READMEs.
- **Data Portfolio Strategy** — Presenting the Capstone project on GitHub and LinkedIn.
- **Continuous Learning** — Keeping up with Apache Iceberg, modern data stack, and AI trends.
