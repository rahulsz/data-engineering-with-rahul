## Overview
Pandas is the backbone of Python-based data engineering. Learn to read, clean, transform, and analyze tabular data using DataFrames and Series.

## Key Concepts
- **Series vs DataFrame** — 1D labeled array vs 2D labeled table structure.
- **Reading Data** — `pd.read_csv()`, `pd.read_json()`, `pd.read_excel()` for ingestion.
- **Filtering & Selection** — `.loc[]`, `.iloc[]`, boolean indexing for precise access.
- **GroupBy & Aggregation** — Split-apply-combine for summary statistics.
- **Handling Missing Data** — `.isna()`, `.fillna()`, `.dropna()` strategies.

## Code Examples

```python
import pandas as pd

# Read and inspect
df = pd.read_csv("shipments.csv")
print(df.shape)
print(df.dtypes)
```

```python
# Clean and transform
df["delivery_date"] = pd.to_datetime(df["delivery_date"])
df["cost"] = df["cost"].fillna(0)

# Aggregate by carrier
summary = df.groupby("carrier").agg(
    total_cost=("cost", "sum"),
    avg_cost=("cost", "mean"),
    shipment_count=("shipment_id", "count")
).reset_index()

print(summary)
```

```python
# List comprehension vs Pandas vectorization
# Slow: [row["cost"] * 1.1 for _, row in df.iterrows()]
# Fast:
df["cost_with_tax"] = df["cost"] * 1.1
```
